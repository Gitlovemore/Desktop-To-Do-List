# FloatingTodo 桌面悬浮待办清单

一个简洁的桌面悬浮待办清单工具。

## 下载后如何使用

### 1. 安装 Node.js
如果没有安装 Node.js，先安装 `node-v24.14.0-x64.msi`

### 2. 安装依赖
解压文件后，在命令行进入目录，运行：
```
npm install
```

### 3. 启动应用
双击 `启动.vbs` 即可运行

## 文件说明

| 文件 | 说明 |
|------|------|
| node-v24.14.0-x64.msi | Node.js 安装包（可选） |
| 启动.vbs | 双击启动应用 |
| main.js | 主程序入口 |

## 功能特点

- 悬浮桌面，不遮挡其他窗口
- 点击穿透，不影响正常工作
- 拖拽移动位置
- 开机自启（可选）
- 右键托盘菜单操作

## 常见问题

**Q: 双击启动.vbs 报错怎么办？**
A: 确保 Node.js 已正确安装，或直接运行：
```
npm start
```
